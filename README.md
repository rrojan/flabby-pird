# flabby-pird
Flabby Pird: A clone (of sorts) of the highly popular game Flappy Bird 

Made in PyGame.



## Requirements:
- pygame 2.0.1


## Installation:
1. `git clone https://github.com/rrojan20/flabby-pird.git`
2. `cd` into folder, then `pip3 install -r requirements.txt`
3. `python3 flabby_pird.py`
